# claude-scripts

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

A collection of utilities for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) â€” hooks, custom slash commands, MCP configs, and helper scripts.

## Folder Structure

| Directory    | Purpose                                         |
| ------------ | ----------------------------------------------- |
| `hooks/`     | Claude Code hooks (pre/post tool-call scripts)  |
| `commands/`  | Custom slash-command prompts (`.md` files)       |
| `mcp/`       | MCP server configs and related scripts           |
| `scripts/`   | General-purpose helper scripts                   |

## Getting Started

1. Clone the repo:

   ```bash
   git clone <repo-url>
   cd claude-scripts
   ```

2. Browse the directories above and copy or symlink the utilities you need into your own Claude Code setup.
